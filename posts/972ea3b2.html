    旅行记录之开封

    标签： 无 分类： 未分类 创建时间：2025-10-08 07:10:14 更新时间：2026-05-27 11:42:29

    前言

    2025年国庆假期，从临沂开车自驾游去了开封，3号出发的，路上其实也没有多少人。中午12点出发，开了一路，差不多5个小时左右吧。

    1.西司夜市

    从临沂出发，到了开封，已经快要五点了，什么景点也都没有了，于是就直接去了西司夜市，开封这个城市，倒是电动车挺多的，都是共享的电动车。去了夜市，排队买了那个 ”张三炒凉粉“，真是一点都不好吃，我都不知道这个评论说好吃的在哪里评论的。排队买了那个杏仁茶，都是现磨的粉，谁家的不一样吗？为什么一定要 “周记杏仁茶” 呢？我也不太明白，就是这么多人排队。还有那个 “羊肉炕馍”，我觉得不是特别的难吃，但是没有啥味道。喝了一杯 “大台北” 的双皮奶，也就是那个味道吧。一晚上都在排队，可是最后竟然好吃的是那个不用排队的 “夹鸡腿” 的馍。

    因为十一期间，不知道为啥，房间特别的贵，而且都是订满的情况，最后花了234块钱，找了一个小酒店，就是居民楼改的，说实话，里面还真是不咋地，房间特别小，连个桌子都没有，还挺贵。车子都只能停在路边，开封这个城市，倒是大家都在路边随便停车，不晓得是不是因为十一不贴条，还有就是他们本身的地下停车场就少。

    2.开封博物馆

    因为下雨，我们起的也晚，出门的时候，已经快要中午十二点了，刚好附近有一个 “宋园灌汤包”，于是就去排队吃早中饭。人也很多，又是一个多小时，等我们吃上饭，已经13点30多了，点了一个灌汤包，一个鳕鱼，都不知道为什么会有人推荐这个咸鱼丝，小红书可真是太乱了啊。还有一个铁板牛肉，这个现做的，用石头炒的，也就算是中规中矩吧。我觉得最大的坑就是这个鱼了，还真是不推荐。吃完饭已经两点多了，本来要去铁塔的，后来觉得离开封博物馆比较近，就去了开封博物馆。

    开封博物馆，看了新石器的一些介绍，就是春秋战国大梁都城，还有就是唐宋时期，感觉到了朱温的时候，这后面的几个就没有什么历史了。还有就是宋金时期的，介绍了两个展馆，感觉也没有什么东西。但是我还是逛了两个多小时，一直到了16点40分了，人家准备闭馆了，才出来，准备去下一个地方。

    开车去了铁塔，本来以为通票可以玩三天的，结果被告知，国庆期间只能玩一天，不能玩三天了，我还刷了铁塔的票，结果只能退了，再买明天的。没有地方去了，只能去下一个地方了，于是去了鼓楼夜市。

    3.鼓楼夜市

    鼓楼夜市，也是一个小吃街，就在鼓楼周边有很多摆摊的地方，我都怀疑是不是全开封的本地人都来摆摊了呢？买了点臭豆腐吃了，沿着鼓楼夜市小路，走到了书店街。然后去了一个卖烤鸭的地方，人也非常的多，排队等着买了半个烤鸭，还是外带走的，也很长时间，还是等了半个小时。拿着东西去了车上吃了，枣木烤鸭我在杭州也吃过，后来那家倒闭了，大家都是一个味道。吃完了东西，今天哪里也去不了了，就开车去了清明上河园，准备看看那里有没有什么东西。于是就在清明上河园和翰林园

    4.铁塔

    为了一天都能玩遍四个景区，于是我们今天早出发了，11点多就到了铁塔，景区其实挺小的，不过就是有一个互动小游戏，要来回的跑，每一个NPC都有自己的任务，最后完成了任务可以拿到一张银票，人很多，我都不记得到底有哪些人了。来来回回的，最后也只转了210两银票，逛遍了整个的景区，已经两个小时了。

    5.万岁山

    玩苏珊还是挺大的

    6.城墙

    其实没啥可看的，比起西安的城墙，可真是差远了，而且城墙一半已经变成了土坡，没有墙只有后面安装的护栏，靠近万岁山的一侧才加了几个砖石结构的墙。

    7.翰林

    从万岁山南门出来，骑了一辆自行车，直接去了

    8.山陕甘会馆

    赵家四味菜，让我又排了两个小时的队，后面的人根本没有吃到，去了 “山陕甘” 会馆，花了23块钱的门票，其实就是一个大院子，其他的啥都没有。

    从会馆出来，本来要去 “州桥和汴河遗址”，到了门口，被告知要98的门票，我不认为它能比秦始皇的墓要有价值的多，但是还要这么多钱，于是我就果断的不去了，毕竟这玩意真是太坑人了。

    9.繁塔

    花了15块钱，就是一个塔，里面真的啥都没有，还离景区特别的园，要是没有时间，我不建议去，旁边的禹王台是一个公园，我觉得还是不去为好，什么小红书推荐，我强烈不推荐。

    10.金明池

    早上起床之后，就去附近找了一个早餐，吃了逍遥胡辣汤，然后骑车去了大梁门，吃了 “西口郑羊双肠”，我觉得一般吧。然后骑车回去，开车去了金明池，金明池遗址是一个湖，有几个宫殿建筑，是党员政治教育基地，也不让进了，倒是还是能。

    11.铁塔

    因为有人说可以三天重复进万岁山，于是我们把车停在了大梁门，然后骑车去了万岁山，被告知就是十一的票只能进一天，但是中秋以后的票可以连玩三天，我们是中秋前一天进去的，就没办法玩了，可真是令人难过，没想到竟然变的这么快。早知道直接买清明上河园的票，然后中秋再买其他的票就好了。

    后记

    2025年10月国庆期间，10月6号晚上开车从开封回到临沂，其实也不是很堵的，堵在路上的，或许就是那些热门的城市吧，还有就是京沪京广这些高速。我觉得开封能逛的也就是万岁山，清明上河园我没有进去，估计是和万岁山一样的套路，都是看景玩游戏，其实城墙啊、铁塔、金明池、翰林什么的，这些都可以不用去了，玩三天也够了其实，但是我们玩了五天，可真是厉害。

    isNaN 和 Number.isNaN 函数的区别？

    1.函数 isNaN 接收参数后，会尝试将这个参数转换为数值，任何不能被转换为数值的的值都会返回 true，因此非数字值传入也会返回 true ，会影响 NaN 的判断。

    2.函数 Number.isNaN 会首先判断传入参数是否为数字，如果是数字再继续判断是否为 NaN ，不会进行数据类型的转换，这种方法对于 NaN 的判断更为准确。

    为什么0.1+0.2 ! == 0.3，如何让其相等?

    一个直接的解决方法就是设置一个误差范围，通常称为“机器精度”。对JavaScript来说，这个值通常为2-52，在ES6中，提供了Number.EPSILON属性，而它的值就是2-52，只要判断0.1+0.2-0.3是否小于Number.EPSILON，如果小于，就可以判断为0.1+0.2 ===0.3。

    == 操作符的强制类型转换规则？

    1.首先会判断两者类型是否**相同，**相同的话就比较两者的大小。

    2.类型不相同的话，就会进行类型转换。

    3.会先判断是否在对比 null 和 undefined，是的话就会返回 true。

    4.判断两者类型是否为 string 和 number，是的话就会将字符串转换为 number。

    5.判断其中一方是否为 boolean，是的话就会把 boolean 转为 number 再进行判断。

    6.判断其中一方是否为 object 且另一方为 string、number 或者 symbol，是的话就会把 object 转为原始类型再进行判断。
